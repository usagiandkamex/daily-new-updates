# 2026/04/09 テクニカル雑談（午前）

## 1. Microsoft

### Windows 11、コントロールパネル廃止へ

**要約**  
Microsoftは、Windows 11において「コントロールパネル」を完全に廃止し、「設定アプリ」へ統合すると発表しました。この変更により、すべての設定が一元化され、ユーザーインターフェースがよりシンプルで直感的になることを目指しています。

**影響**  
エンジニアにとっては、スクリプトや自動化ツールの修正が必要になる可能性があります。特に、既存のプロセスがコントロールパネルを利用している場合、設定アプリへの移行を考慮する必要が出てきます。

**参考リンク**  
[Windows 11、コントロールパネル廃止へ](https://ascii.jp/elem/000/004/388/4388318/)

### Azure MigrateがAzure Filesの評価をサポート

**要約**  
Azure MigrateでAzure Filesの評価が可能になりました。この機能により、オンプレミスのSMBやNFSファイルの移行計画が効率化されます。

**影響**  
オンプレミスからクラウドへの移行を計画しているエンジニアにとって、Azure Migrateの新機能は、移行プロセスの計画と実行を大幅に簡素化します。

**参考リンク**  
[Azure Files assessments are now available using Azure Migrate](https://ascii.jp/elem/000/004/017/4017782/2/)
