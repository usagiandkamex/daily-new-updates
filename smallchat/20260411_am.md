## 4. クラウド（AWS / GCP / OCI）

### Recent AWS CLI s3/streaming change

**要約**: AWS CLI の最近の変更により、S3 のストリーミング操作が影響を受けるようになったとの報告があります。この変更は、特定のユースケースでの操作性に影響を与える可能性があるため、多くのエンジニアが議論しています。

**影響**: S3 を使用したストリーミング操作を行っているプロジェクトに影響を与える可能性があります。CLI を使用するスクリプトやワークフローを再確認し、必要に応じて調整が必要です。

**参考リンク**: [Recent AWS CLI s3/streaming change](https://aws.amazon.com/jp/blogs/news/amazon-kinesis-data-firehose-custom-prefixes-for-amazon-s3-objects/)
