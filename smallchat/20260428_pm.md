### GitHub Copilot is moving to usage-based billing

**要約**: GitHub Copilotが従量課金モデルに移行します。2026年6月1日以降、Copilotの利用はGitHub AI Creditsを消費する形になります。

**影響**: 開発者にとって、使用量に応じた料金体系が導入されるため、コスト管理が重要になります。一方で、利用頻度が少ないユーザーにとってはコスト削減につながる可能性があります。

**リンク**: [GitHub Copilot is moving to usage-based billing](https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/)
