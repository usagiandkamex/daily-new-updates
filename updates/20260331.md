# 2026/03/31 デイリーアップデート

## 1. Azure アップデート情報

### Ephemeral OS Disk with Full Cachingがパブリックプレビュー開始

**要約**: Azureの仮想マシン（VM）および仮想マシンスケールセット（VMSS）向けに、Ephemeral OS Diskのフルキャッシング機能がパブリックプレビューとして利用可能になりました。この機能により、OSディスクのパフォーマンスが大幅に向上します。

**影響**: このアップデートにより、Azureの利用者はより高速で信頼性の高いOSディスクパフォーマンスを享受できるようになります。特に、高速なディスクI/Oを必要とするワークロードにとって有益です。

**参考リンク**: [Ephemeral OS Disk with full caching](https://azure.microsoft.com/updates?id=559322)

### Azure Red Hat OpenShiftがインドネシア中央リージョンで一般提供開始

**要約**: Azure Red Hat OpenShift（ARO）がインドネシア中央リージョンで一般提供を開始しました。これにより、インドネシアの顧客はAROを利用して、クラウドネイティブアプリケーションを効率的に開発・展開することが可能になります。

**影響**: インドネシア市場でのAzureの存在感が強化され、現地企業のクラウド移行が加速することが期待されます。

**参考リンク**: [Azure Red Hat OpenShift in Indonesia Central](https://azure.microsoft.com/updates?id=559552)

### Azure Premium SSD v2が南インドで利用可能に

**要約**: Azure Premium SSD v2が南インドリージョンで利用可能になりました。この次世代の汎用SSDディスクは、より高いパフォーマンスとコスト効率を提供します。

**影響**: 南インドリージョンのユーザーは、より高性能なストレージオプションを利用できるようになり、アプリケーションのパフォーマンス向上が期待されます。

**参考リンク**: [Azure Premium SSD v2 Disk](https://azure.microsoft.com/updates?id=559526)

### Azure MonitorのPrometheusサイドカー機能が2027年に廃止予定

**要約**: Azure MonitorのPrometheus用サイドカー機能が2027年3月31日に廃止される予定です。これに伴い、ユーザーは代替ソリューションへの移行を検討する必要があります。

**影響**: 現在この機能を利用しているユーザーは、廃止期限までに新しいソリューションを導入する必要があります。移行計画の策定が求められます。

**参考リンク**: [Deprecation of sidecar for remote-write for Azure Monitor](https://azure.microsoft.com/updates?id=550519)

### Azure Cosmos DB for PostgreSQLが2029年に廃止予定

**要約**: Azure Cosmos DB for PostgreSQLが2029年3月31日に廃止される予定です。ユーザーはAzure Database for PostgreSQL Elastic Clusterへの移行を検討する必要があります。

**影響**: 長期的な計画を立てる必要があり、特にデータベースを利用している企業にとっては、移行に伴うコストやリソースの再配置が課題となります。

**参考リンク**: [Azure Cosmos DB for PostgreSQL Retirement](https://azure.microsoft.com/updates?id=556085)
