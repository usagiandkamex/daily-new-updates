# 2026/04/02 デイリーアップデート

## 1. Azure アップデート情報

### Service FabricのWindows Server 2022サポート終了が発表

**要約**: Microsoftは、Service FabricのWindows Server 2022サポートを2027年3月31日で終了すると発表しました。ユーザーはWindows Server 2025へのアップグレードを推奨されています。

**影響**: サポート終了により、セキュリティリスクが増加する可能性があるため、企業は早急にアップグレード計画を立てる必要があります。

**参考リンク**: [詳細はこちら](https://azure.microsoft.com/updates?id=558247)

### Azure Data Boxが新機能を追加

**要約**: Azure Data BoxがAzure Files Provisioned v2のサポートを開始し、さらに自動でセキュア消去証明書を生成する機能を追加しました。

**影響**: データ移行がより効率的かつ安全に行えるようになり、企業のコンプライアンス対応が強化されます。

**参考リンク**: [詳細はこちら](https://azure.microsoft.com/updates?id=559767)

### Azure App Configurationがクライアントサイド設定を強化

**要約**: Azure App ConfigurationがAzure Front Doorとの統合により、クライアント側での設定管理を大規模に行える機能をプレビューリリースしました。

**影響**: アプリケーションのパフォーマンスとセキュリティが向上し、開発者の負担が軽減されます。

**参考リンク**: [詳細はこちら](https://azure.microsoft.com/updates?id=537234)

### Azure NetApp Filesが新しいQoS機能を追加

**要約**: Azure NetApp Filesが「cool access」を有効にしたQoSアップデートを発表。これにより、コストとパフォーマンスのバランスが改善されます。

**影響**: ストレージコストを削減しつつ、パフォーマンスを維持したい企業にとって有益な選択肢となります。

**参考リンク**: [詳細はこちら](https://azure.microsoft.com/updates?id=559504)

### Azure AI Languageの一部機能が2027年に終了予定

**要約**: Azure AI Languageの8つの機能が2027年3月20日をもって終了することが発表されました。対象機能には感情分析やキーフレーズ抽出が含まれます。

**影響**: これらの機能を利用している企業は、代替ソリューションを検討する必要があります。

**参考リンク**: [詳細はこちら](https://azure.microsoft.com/updates?id=557394)
