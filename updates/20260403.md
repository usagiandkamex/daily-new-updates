# 2026/04/03 デイリーアップデート

## 1. Azure アップデート情報

### Azure App ServiceとAzure FunctionsのWindows上でのPythonサポートが2027年3月31日に終了

**要約**: Microsoftは、Azure App ServiceおよびAzure FunctionsのWindows環境におけるPythonサポートを2027年3月31日に終了することを発表しました。この変更により、PythonアプリケーションをWindows上でホストしているユーザーは、他のプラットフォームへの移行を検討する必要があります。

**影響**: 開発者や企業は、Pythonアプリケーションを引き続き利用するために、Azure Linux環境や他のクラウドサービスへの移行を計画する必要があります。これにより、移行コストや運用の見直しが求められる可能性があります。

**参考リンク**: [Retirement: Python support for Azure App Service and Azure Functions on Windows will be retired on March 31, 2027](https://azure.microsoft.com/updates?id=558027)

### Azure Blob StorageのAzureDnsZoneエンドポイントが2027年3月に廃止予定

**要約**: Microsoftは、Azure Blob StorageアカウントにおけるAzureDnsZoneエンドポイントのプレビュー提供を終了し、2027年3月に完全廃止する予定です。これにより、現在このエンドポイントを利用している顧客は、他のオプションに移行する必要があります。

**影響**: この変更は、Azure Blob Storageを利用する企業や開発者にとって、設定変更や移行計画が必要となる可能性があります。特に、エンドポイントの変更に伴うダウンタイムや追加の設定作業が発生する可能性があります。

**参考リンク**: [Retirement: AzureDnsEndpoints for Azure storage accounts will be retired March 2027](https://azure.microsoft.com/updates?id=558276)
