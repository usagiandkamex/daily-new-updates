## 1. Azure アップデート情報

### Retirement: Azure Reserved Virtual Machines Instances for select VM series

**要約**: 2026年7月1日より、Av2、Amv2、Bv1、D、Ds、Dv2などの特定のVMシリーズにおける1年契約のAzure Reserved Virtual Machines Instancesの購入および更新が利用できなくなります。

**影響**: 対象となるVMシリーズを利用しているユーザーは、今後の更新や新規購入ができなくなるため、代替オプションを検討する必要があります。

**リンク**: [Retirement: Azure Reserved Virtual Machines Instances for select VM series](https://azure.microsoft.com/ja-jp/updates?id=560948)

---

### [In preview] Public Preview: Application Gateway for Containers managed add-on + AKS Automatic

**要約**: Application Gateway for ContainersおよびAzure Kubernetes Service (AKS) 用のApplication Gateway for Containers Add-onがパブリックプレビューとして利用可能になりました。この機能は、コンテナ環境でのアプリケーションゲートウェイの管理を簡素化します。

**影響**: AKSを利用する開発者や運用チームにとって、コンテナ環境でのトラフィック管理がより効率的になります。

**リンク**: [Public Preview: Application Gateway for Containers managed add-on + AKS Automatic](https://azure.microsoft.com/ja-jp/updates?id=558403)

### [Launched] Generally Available: Azure Elastic SAN support for AVS Gen2 Private Cloud

**要約**: Azure Elastic SANがAVS（Azure VMware Solution）Gen2 Private Cloudで一般提供されました。この新機能により、接続性とパフォーマンスが向上し、設定が簡素化されます。

**影響**: AVS Gen2 Private Cloudを利用しているユーザーは、よりシンプルな接続と高性能なストレージオプションを活用できます。

**リンク**: [Generally Available: Azure Elastic SAN support for AVS Gen2 Private Cloud](https://azure.microsoft.com/ja-jp/updates?id=560909)

---

### [Launched] Generally Available: Single Volume Snapshots on Azure Elastic SAN

**要約**: Azure Elastic SANが単一ボリュームのスナップショットをサポートするようになりました。この機能により、SANボリュームの増分バックアップやポイントインタイムの復元が可能になります。

**影響**: データ保護とバックアッププロセスが簡素化され、より柔軟なデータ管理が可能になります。

**リンク**: [Generally Available: Single Volume Snapshots on Azure Elastic SAN](https://azure.microsoft.com/ja-jp/updates?id=560899)

---

### [Launched] Generally Available: AVS Support for AV64 SKU on Azure Elastic SAN

**要約**: Azure Elastic SANがAV64 SKUをサポートするようになり、Azure VMware Solution (AVS) における高スケール・高性能なストレージオプションが利用可能になりました。

**影響**: AVSユーザーは、より大規模で高性能なストレージニーズに対応できるようになります。

**リンク**: [Generally Available: AVS Support for AV64 SKU on Azure Elastic SAN](https://azure.microsoft.com/ja-jp/updates?id=560894)
