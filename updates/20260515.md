## 1. Azure アップデート情報

### [Launched] Generally Available: Support for workloads with large files in Azure NetApp Files

**要約**: Azure NetApp Files が、Azure VMware Solution (AVS) の仮想マシンで使用される大容量 VMDK ファイルを含む、大容量ファイルを使用するワークロードのシームレスな移行と運用をサポートする機能を一般提供開始しました。

**影響**: 大容量ファイルを扱うワークロードを Azure NetApp Files に移行する際の効率性が向上し、特に AVS 環境での運用が簡素化されます。

**リンク**: [Generally Available: Support for workloads with large files in Azure NetApp Files](https://azure.microsoft.com/ja-jp/updates?id=561722)

---

### [Launched] Generally Available: Managed Identity Support for Azure Files SMB Is now GA

**要約**: Azure Files が Managed Identity を使用した SMB アクセスをサポートする機能を一般提供開始しました。これにより、アプリケーションやサービスが静的な資格情報を保存せずに認証を行うことが可能になります。

**影響**: セキュリティが向上し、資格情報管理の負担が軽減されます。特に、動的な認証を必要とする環境での利用が容易になります。

**リンク**: [Generally Available: Managed Identity Support for Azure Files SMB Is now GA](https://azure.microsoft.com/ja-jp/updates?id=562350)
