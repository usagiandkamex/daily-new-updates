### [Launched] Update: Microsoft Foundry built-in RBAC role naming and enhancements

**要約**: Microsoft Foundry の組み込み RBAC (ロールベースのアクセス制御) ロール名が、製品ブランドに合わせて更新されました。これにより、「Azure AI アカウントオーナー」などの役割名が変更され、より直感的でわかりやすい命名規則が採用されています。

**影響**: 既存の Microsoft Foundry ユーザーは、新しいロール名に慣れる必要がありますが、機能自体に変更はありません。管理者は、ロール名の変更に伴うドキュメントやプロセスの更新を検討する必要があります。

**リンク**: [Update: Microsoft Foundry built-in RBAC role naming and enhancements](https://azure.microsoft.com/ja-jp/updates?id=562533)

### [Launched] Generally Available: Azure Blob Storage SDK for Rust

**要約**: Azure Blob Storage SDK for Rust が一般提供開始されました。この SDK を使用することで、Rust 言語を使用して Azure Blob Storage アカウントに接続し、データ操作を行うことが可能になります。

**影響**: Rust を使用している開発者にとって、Azure Blob Storage との統合が容易になり、効率的なデータ管理が可能になります。特に、Rust を採用しているプロジェクトにおいて、Azure のストレージ機能を活用する新たな選択肢が増えました。

**リンク**: [Generally Available: Azure Blob Storage SDK for Rust](https://azure.microsoft.com/ja-jp/updates?id=562516)
