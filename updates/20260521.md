### [Launched] Generally Available: langchain-azure-cosmosdb python package for Azure Cosmos DB

**要約**: Azure Cosmos DB 向けの新しい Python パッケージ「langchain-azure-cosmosdb」が一般提供されました。このパッケージを使用することで、LangChain および LangGraph を統合し、AIやエージェントアプリケーションをより迅速に本番環境向けに構築することが可能です。

**影響**: Azure Cosmos DB を利用している開発者は、この新しいパッケージを活用することで、AIアプリケーションの開発プロセスを効率化し、生産性を向上させることができます。

**要約**: Azure Storage Mover が Blob コンテナー間のデータ転送をサポートするようになり、異なるリージョンやサブスクリプション間でのデータ移動が簡単になりました。

**影響**: 大量のデータを移行する必要がある企業や組織にとって、データ移行の効率性と柔軟性が向上します。

**リンク**: [Generally Available: Azure Storage Mover Blob-to-Blob migration](https://azure.microsoft.com/ja-jp/updates?id=562753)

**影響**: ユーザーはデータ移行のタイミングを柔軟に管理できるようになり、運用効率が向上します。

**リンク**: [Generally Available: Schedule one-time or recurring migrations with Azure Storage Mover](https://azure.microsoft.com/ja-jp/updates?id=562622)

### [Launched] Generally Available: site-to-site VPN connections with certificate authentication

**要約**: Azure Site-to-Site VPN がデジタル証明書認証をサポートするようになりました。これにより、従来の事前共有キー（PSK）モデルに代わるセキュアな認証方法が提供されます。

**影響**: セキュリティ要件が高い環境での VPN 接続において、より安全な認証オプションを利用できるようになります。

**リンク**: [Generally Available: site-to-site VPN connections with certificate authentication](https://azure.microsoft.com/ja-jp/updates?id=562705)

**要約**: Azure Event Grid において、サブスクリプション識別子がパブリックプレビューとして利用可能になりました。この機能により、どのサブスクリプションがメッセージ配信をトリガーしたかを識別することが可能になります。
