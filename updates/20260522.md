# 2026/05/22 デイリーアップデート

## 1. Azure アップデート情報

### [Launched] Generally Available: Entra-only identities with Azure Files

**要約**: Azure Filesが、SMBアクセスのためのEntra-onlyアイデンティティの一般提供を開始しました。これにより、組織はクラウドベースの認証を使用してファイル共有に安全にアクセスできるようになります。

**影響**: このアップデートは、オンプレミスのActive Directoryを使用せずに、Azure Entra IDを活用してファイル共有のアクセス管理を簡素化したい企業にとって重要です。

**リンク**: [Generally Available: Entra-only identities with Azure Files](https://azure.microsoft.com/ja-jp/updates?id=562359)

---

### [Launched] Generally Available: Azure NetApp Files object REST API

**要約**: Azure NetApp Filesが、オブジェクトREST API（S3互換REST API）の一般提供を開始しました。この新機能により、従来のファイルベースのストレージと最新のクラウドサービスとのギャップを埋めることが可能になります。

**影響**: このアップデートは、Azure NetApp Filesを利用している企業が、クラウドネイティブなアプリケーションと統合しやすくなるため、ストレージ管理の柔軟性が向上します。

**リンク**: [Generally Available: Azure NetApp Files object REST API](https://azure.microsoft.com/ja-jp/updates?id=562254)

---

### Retirement: TLS 1.0 and TLS 1.1 in Azure App Service, Azure Functions, and Azure Logic Apps

**要約**: Azureはセキュリティ向上の一環として、Azure App Service、Azure Functions、Azure Logic AppsにおけるTLS 1.0およびTLS 1.1の接続を受け付けなくなることを発表しました。

**影響**: この変更は、TLS 1.2以降を使用する必要があるため、古いTLSバージョンを使用しているアプリケーションの更新が必要です。セキュリティの向上が期待されます。

**リンク**: [Retirement: TLS 1.0 and TLS 1.1 in Azure App Service, Azure Functions, and Azure Logic Apps](https://azure.microsoft.com/ja-jp/updates?id=557852)

---

### [Launched] Generally Available: User Groups and IP address pools for P2S connections

**要約**: VPN GatewayのPoint-to-Site接続において、ユーザーグループとIPアドレスプールの機能が一般提供されました。この機能により、リモートユーザーごとに異なるIPアドレスプールを割り当てることが可能になります。

**影響**: このアップデートは、VPN接続の管理をより柔軟にし、セキュリティとネットワーク管理の効率化を実現します。

**リンク**: [Generally Available: User Groups and IP address pools for P2S connections](https://azure.microsoft.com/ja-jp/updates?id=564460)

---

### [In preview] Public Preview: TLS/SSL certificate support for Azure Functions Flex Consumption

**要約**: Azure Functions Flex Consumptionで、サイトスコープのTLS/SSL証明書モデルがパブリックプレビューとして提供開始されました。この機能は、Flex Consumptionが利用可能なすべてのリージョンで利用可能です。

**影響**: このプレビューは、Azure Functionsを利用する開発者にとって、セキュリティと柔軟性を向上させる新しい選択肢を提供します。

**リンク**: [Public Preview: TLS/SSL certificate support for Azure Functions Flex Consumption](https://azure.microsoft.com/ja-jp/updates?id=562808)
