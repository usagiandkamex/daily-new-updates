## 1. Azure アップデート情報

### [In preview] Public Preview: Azure Virtual Network Manager integration with Virtual WAN

**要約**: Azure Virtual Network Manager が Virtual WAN と統合され、パブリックプレビューとして利用可能になりました。この統合により、Azure Virtual WAN ハブを Azure Virtual Network Manager のハブとして使用できるようになります。

**影響**: この新機能により、ネットワーク管理がさらに効率化され、Azure Virtual WAN ハブを活用したネットワーク設計が容易になります。特に、大規模なネットワーク環境を管理する企業にとって、柔軟性とスケーラビリティが向上します。

**リンク**: [Public Preview: Azure Virtual Network Manager integration with Virtual WAN](https://azure.microsoft.com/ja-jp/updates?id=564478)

---

### [Launched] Generally Available: Virtual network flow logs connector with Microsoft Sentinel

**要約**: Azure Virtual Network Flow Logs Connector が Microsoft Sentinel と統合され、一般提供が開始されました。この機能により、ネットワークトラフィックデータをシームレスにエクスポートし、詳細な分析が可能になります。

**影響**: セキュリティ運用チームは、ネットワークトラフィックの可視性を向上させ、脅威の検出と応答を迅速化できます。Microsoft Sentinel を利用することで、ネットワークログの監視と分析が一元化されます。

**リンク**: [Generally Available: Virtual network flow logs connector with Microsoft Sentinel](https://azure.microsoft.com/ja-jp/updates?id=564689)
