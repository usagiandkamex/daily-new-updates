**要約**: Azure Premium SSD v2 ディスクが、非ゾーン対応の単一インスタンス Azure 仮想マシンをサポートするようになりました。この機能は、特定の Azure リージョンで利用可能です。

**リンク**: [Microsoft Foundry agent security capabilities in Microsoft Defender for Cloud are transitioning to a Microsoft Agent 365 license](https://azure.microsoft.com/ja-jp/updates?id=565171)

### [In preview] Public Preview: Azure Site Recovery support for Linux Azure VMs with NVMe disk controllers

**要約**: Azure Site Recovery が、NVMe ディスクコントローラーを搭載した Linux Azure 仮想マシンのレプリケーションと災害復旧をサポートするようになりました。

**リンク**: [Azure Site Recovery support for Linux Azure VMs with NVMe disk controllers](https://azure.microsoft.com/ja-jp/updates?id=565103)

### [Launched] Generally Available: Maintenance control for Azure Database for PostgreSQL - reschedule, apply on demand, view and download

**要約**: Azure Database for PostgreSQL Flexible Server において、メンテナンスコントロール機能が一般提供開始されました。この機能により、メンテナンスの再スケジュール、オンデマンド適用、ビューおよびダウンロードが可能になります。

**影響**: データベース管理者は、メンテナンススケジュールを柔軟に管理できるようになり、システムの可用性と運用効率が向上します。

**リンク**: [Maintenance control for Azure Database for PostgreSQL - reschedule, apply on demand, view and download](https://azure.microsoft.com/ja-jp/updates?id=563756)

### [In preview] Public Preview: Azure Managed Redis now supports Entra ID based RBAC for data management

**要約**: Azure Managed Redis が、Entra ID をベースとしたロールベースアクセス制御 (RBAC) をサポートするようになりました。これにより、データ管理におけるアクセス権限をより細かく制御できます。

**影響**: Redis データの読み取り、書き込み、管理権限をより厳密に制御する必要があるユーザーにとって、セキュリティと運用効率の向上が期待されます。

**リンク**: [Azure Managed Redis now supports Entra ID based RBAC for data management](https://azure.microsoft.com/ja-jp/updates?id=564873)
