## 1. Azure アップデート情報

### [Launched] Generally Available: Microsoft Entra server principals on Azure SQL Database

**要約**: Microsoft Entra サーバープリンシパル（ログイン）が Azure SQL Database で一般提供開始されました。この機能により、`CREATE LOGIN ... FROM EXTERNAL PROVIDER` を使用して、外部プロバイダーからログインを作成することが可能になります。

**影響**: Azure SQL Database を利用しているユーザーは、Microsoft Entra を活用した認証の柔軟性が向上し、セキュリティと管理性が強化されます。

**リンク**: [タイトル](https://azure.microsoft.com/ja-jp/updates?id=565154)

---

### Retirement: Azure Synapse Link for Azure Cosmos DB NoSQL

**要約**: 2026年3月31日をもって、新規の Azure Cosmos DB NoSQL アカウントで Azure Synapse Link を有効にすることができなくなります。既存の顧客は引き続き完全にサポートされます。

**影響**: 新規アカウントでの Azure Synapse Link の利用が制限されるため、これから Cosmos DB NoSQL を導入するユーザーは代替手段を検討する必要があります。

**リンク**: [タイトル](https://azure.microsoft.com/ja-jp/updates?id=558560)

---

### Retirement: Azure VPN Client for Linux (Preview) will be retired on August 31, 2026

**要約**: Azure VPN Client for Linux (プレビュー版) は、2026年8月31日をもって廃止されます。このクライアントは一般提供 (GA) に進む予定がないため、廃止が決定されました。

**影響**: Linux 環境で Azure VPN Client を利用しているユーザーは、廃止期限までに代替ソリューションを検討する必要があります。

**リンク**: [タイトル](https://azure.microsoft.com/ja-jp/updates?id=565393)
