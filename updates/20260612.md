## 1. Azure アップデート情報

### Retirement: Av2-series, F-series, Fs-series, Fsv2-series, G-series, Gs-series, and Lsv2-series Virtual Machines for Azure Batch pools

**要約**: Azure Compute は、2028年11月15日に Av2シリーズ、Fシリーズ、Fsシリーズ、Fsv2シリーズ、Gシリーズ、Gsシリーズ、Lsv2シリーズの仮想マシンを Azure Batch プールでの使用からリタイアします。

**影響**: これらの仮想マシンシリーズを使用している Azure Batch プールは、リタイア日以降にこれらのリソースを利用できなくなります。ユーザーは、代替の仮想マシンシリーズへの移行を計画する必要があります。

**リンク**: [Retirement: Av2-series, F-series, Fs-series, Fsv2-series, G-series, Gs-series, and Lsv2-series Virtual Machines for Azure Batch pools](https://azure.microsoft.com/ja-jp/updates?id=564774)

**リンク**: [Retirement: D-series, Ds-series, Dv2-series, Dsv2-series, and Ls-series Virtual Machines for Azure Batch pools](https://azure.microsoft.com/ja-jp/updates?id=564772)

### Retirement: Azure Load Balancer Inbound NAT rule version 1 for Azure VMSS (aka Inbound NAT Pools)

**要約**: 以前のアナウンスメントに基づき、Azure Load Balancer Inbound NATルールバージョン1のリソースがリタイアされます。これには、単一インスタンスとAzure VMSSのInbound NAT Poolsが含まれます。
