## 1. Azure アップデート情報

### [Launched] Generally Available: Client-side data integrity protections in Azure Blob Storage

**要約**: Azure Blob Storage がクライアント側でのデータ整合性保護機能を正式にリリースしました。この機能は、従来の HTTP 標準に基づく MD5 サポートに加え、CRC64-NVME サポートを追加することで、データの整合性をさらに強化します。

**影響**: このアップデートにより、Azure Blob Storage を使用するユーザーは、データ転送中の整合性チェックをより簡単に実施できるようになります。これにより、データの信頼性が向上し、特にデータ転送の安全性を重視する企業にとって有益です。

**リンク**: [Generally Available: Client-side data integrity protections in Azure Blob Storage](https://azure.microsoft.com/ja-jp/updates?id=566895)

---

### [Launched] Generally Available: Toolboxes in Microsoft Foundry

**要約**: Microsoft Foundry が「Toolboxes」を正式リリースしました。この新機能により、プロンプトエージェントで最も頻繁に使用されるツールを統合エンドポイントから呼び出すことが可能になり、プロンプト外でも利用できるようになります。

**影響**: Toolboxes の一般提供により、開発者はプロンプトエージェントを使用した効率的なツール管理が可能となり、作業プロセスの効率化が期待されます。また、統合エンドポイントの導入により、ツールの利用がより直感的で簡単になります。

**リンク**: [Generally Available: Toolboxes in Microsoft Foundry](https://azure.microsoft.com/ja-jp/updates?id=563481)
