### [Launched] Generally Available: Microsoft Entra ID-based access for Azure Blob Storage SFTP

**要約**: Microsoft Entra ID を使用した Azure Blob Storage SFTP へのアクセスが、すべてのリージョンで一般提供されました。これにより、Entra ID の ID を利用して、セキュアなファイル転送プロトコル (SFTP) を活用することが可能になります。

### [Launched] Generally Available: Support 5x churn in Azure Site Recovery

**要約**: Azure Site Recovery が 1 仮想マシンあたり最大 500 MB/s のデータ変更率 (5 倍の変更率) をサポートするようになりました。この改善により、高い IOPS を必要とするワークロードでも安心して利用できるようになります。

**影響**: このアップデートは、データ量が多い環境や高いパフォーマンスを求める企業にとって、Azure Site Recovery の利用価値をさらに高めるものです。

**リンク**: [タイトル](https://azure.microsoft.com/ja-jp/updates?id=566966)
