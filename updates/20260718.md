# 2026/07/18 デイリーアップデート

## 1. Azure アップデート情報

### [Launched] Generally Available: Azure Functions support for Python 3.14

**要約**: Azure Functions が Python 3.14 を正式にサポートするようになりました。これにより、ローカルで Python 3.14 を使用して関数を開発し、Linux 上の Azure Functions プランにデプロイすることが可能です。既存のアプリケーションを Python 3.14 にアップグレードすることで、最新の機能と改善を活用できます。

**影響**: Python を使用して Azure Functions を開発しているユーザーにとって、最新バージョンへの対応により、より高度な機能やパフォーマンスの向上が期待できます。

**リンク**: [Generally Available: Azure Functions support for Python 3.14](https://azure.microsoft.com/ja-jp/updates?id=567646)

---

### [Launched] Public Preview: Azure Functions Support for PowerShell 7.6

**要約**: Azure Functions が PowerShell 7.6 をサポートするパブリックプレビューを開始しました。これにより、ローカルで PowerShell 7.6 を使用してアプリケーションを開発し、Azure にデプロイすることが可能になります。

**影響**: PowerShell を使用する開発者にとって、最新バージョンのサポートにより、より柔軟で効率的なスクリプト作成とデプロイが可能になります。

**リンク**: [Public Preview: Azure Functions Support for PowerShell 7.6](https://azure.microsoft.com/ja-jp/updates?id=567651)

---

### [Launched] Generally Available: Microsoft Defender security assessments for Azure Database for PostgreSQL Flexible Server

**要約**: Microsoft Defender のクラウドセキュリティポスチャ管理（CSPM）が、Azure Database for PostgreSQL Flexible Server のセキュリティ評価を正式にサポートするようになりました。この機能により、データベースのセキュリティ状態を監視し、潜在的なリスクを特定することができます。

**影響**: Azure Database for PostgreSQL Flexible Server を使用しているユーザーは、セキュリティ評価機能を活用することで、セキュリティリスクを軽減し、より安全なデータベース運用が可能になります。

**リンク**: [Generally Available: Microsoft Defender security assessments for Azure Database for PostgreSQL Flexible Server](https://azure.microsoft.com/ja-jp/updates?id=567527)

---

### [Launched] Generally Available: Encryption in Transit for Azure Files NFS Shares in Azure Kubernetes Service (AKS)

**要約**: Azure Kubernetes Service (AKS) が、Azure Files NFS v4.1 ボリュームにおけるデータ転送時の暗号化（Encryption in Transit, EiT）を正式にサポートするようになりました。この機能は Azure File CSI ドライバーを通じて提供されます。

**影響**: AKS を利用するユーザーにとって、データ転送時のセキュリティが向上し、より安全なクラウドネイティブアプリケーションの運用が可能になります。

**リンク**: [Generally Available: Encryption in Transit for Azure Files NFS Shares in Azure Kubernetes Service (AKS)](https://azure.microsoft.com/ja-jp/updates?id=567787)
